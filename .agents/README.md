# Agents

Набор AI-агентов для разработки проекта cake-template.

## Доступные агенты

| Агент | Описание | Когда использовать |
|-------|----------|-------------------|
| `agent-coder` | Написание кода | Реализация фич, рефакторинг |
| `agent-reviewer` | Ревью кода | Проверка качества, безопасности |
| `agent-planner` | Планирование | Декомпозиция задач, приоритеты |
| `agent-tester` | Тестирование | Unit/integration/e2e тесты |
| `agent-architecture` | Архитектура | Системный дизайн, масштабирование |
| `agent-performance-optimizer` | Оптимизация | Производительность, бутылочные горлышки |
| `agent-security-manager` | Безопасность | Аудит, уязвимости |
| `agent-dev-backend-api` | Backend API | API дизайн, эндпоинты |
| `agent-github-pr-manager` | PR менеджер | Создание/ревью PR |
| `agent-sparc-coordinator` | SPARC координатор | Методология разработки |
| `agent-researcher` | Исследования | Анализ, поиск решений |
| `agent-refinement` | Доработка | Улучшение существующего кода |
| `agent-specification` | Спецификации | Требования, ТЗ |
| `agent-scout-explorer` | Разведка | Изучение кодовой базы |
