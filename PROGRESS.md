# Прогресс проекта cake-template

## Что это
Шаблон продающего сайта для кондитерских инста-магазинов Москвы. Цель — быстро адаптировать под конкретного клиента, показывать как демо и продавать услугу.

## Стек
- Next.js 14+ (App Router)
- Tailwind CSS
- TypeScript
- Framer Motion (анимации)
- Хостинг: Vercel (пока локально)

## Что сделано

### Сессия 1 (2026-04-04)

1. **Создан проект** — Next.js + Tailwind, папка `/Users/ttt/Projects/cake-template/`
2. **Структура сайта:**

## Как адаптировать под нового клиента

1. Открыть `lib/config.ts`
2. Поменять: название, телефон, WhatsApp, Telegram, Instagram, город
3. Поменять каталог тортов (названия, цены, описания, ссылки на фото)
4. Поменять фоновые изображения (images.hero, images.about, images.cta)
5. Задеплоить на Vercel с новым доменом
